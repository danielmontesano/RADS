Comunicación Monoportadora:
- Para ejecutar esta modalidad abrir el archivo "Monoportadora.m". 
  Este programa accederá a las funciones contenidas en la carpeta Monoportadora.
- Seleccionar el modo de receptor que se desea tener, de acuerdo a los descritos en la memoria:
  Establecer la variable "modo" en:
    '1' Recepción no coherente con detección de envolvente
    '2' Recepción heterodina no coherente
    '3' Recepción heterodina y coherente utilizando PLLs sintonizados
- Para cambiar los parámetros que describen las perturbaciones que afectan a la señal,
  acceder al archivo "canalTransmisión.m"


Comunicación de Expansión Espectral:
- Para ejecutar esta modalidad abrir el archivo "ExpansionEspectral.m".
- Ejecutará las funciones de la carpeta Expansion_Espectral
- Puede cambiar el ruido blanco del canal en el propio archivo "ExpansionEspectral.m".
